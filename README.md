## TRIP RESERVATION SYSTEM

A modern responsive web-based **Trip Reservation system** built with **Laravel** and **Bootstrap**.

## features

**User Registration and Login** with role based access
**Trip search and filtering** by destination, price range, and trip title
**Booking Management** for travelers
**Trip package management** for travel agents
**Hotel Profile management** for hotel manager
**Review System** for travelers to review hotels
**Payment Integration** supports online payments
**Admin Dashboard** user management
